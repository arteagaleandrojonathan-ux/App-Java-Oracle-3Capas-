Sistema en Java - Arquitectura 3 Capas con Oracle

En este repositorio comparto mi programa creado con la arquitectura en 3 capas, el proyecto consta de su vista gráfica hecha en Java y maneja todas las conexiones directas con una base de datos en Oracle

Trata Básicamente es un sistema donde separé todo el código en tres niveles para mantener el orden y asegurar que funcione correctamente:
- Capa de Presentación (View): Aquí están todas las vistas gráficas y ventanas con las que interactúa el usuario
- Capa de Negocio: Aquí armé toda la lógica para que el programa procese la información
- Capa de Datos: Es la parte que se encarga exclusivamente de las conexiones de Oracle con Java para hacer consultas, guardar y actualizar la data

Herramientas que usé:
- Lenguaje principal: Java
- Base de Datos: Oracle Database
- Estructura: Arquitectura 3 Capas

Este proyecto lo subo para demostrar cómo manejo las interfaces visuales conectadas a una base de datos relacional y cómo organizo el código para que no esté todo mezclado
